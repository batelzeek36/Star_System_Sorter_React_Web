## BINARY CITATION MODES (NEVER MIX)

### PROVISIONAL (when you can't access full text)
**Use when:** Source exists but you can't access complete text, behind paywall, only preview available
**Required fields:** `location_hint` (chapter/section), `summary` (≤25 words), `citation_status: "provisional"`
**Must NOT include:** `page` or `quote`

### LOCKED (when you have verbatim access)
**Use when:** Full text accessible on Archive.org/sacred-texts.com, can copy exact words
**Required fields:** `page` (or canonical unit like "Tablet IV, lines 93-104"), `quote` (≤25 words verbatim), `citation_status: "locked"`
**Must NOT include:** `location_hint` or `summary`

**CRITICAL:** If source is fully available at sacred-texts.com or archive.org with complete searchable text, you MUST use locked mode. Canonical units (Tablet IV, Session 38.7) count as locked citations.

📚 REQUIRED BIBLIOGRAPHIC FIELDS
═══════════════════════════════════════════════════════════════════════════════

**EVERY source MUST include ALL of:**

1. **title** - Full, exact title (not abbreviated)
2. **author** - Full name. For ancient texts: "Babylonian (Enuma Elish)" or "Anonymous", NOT the translator's name
3. **translator_or_editor** - REQUIRED for ancient/non-English sources. Format: "Full Name (translator)". Omit for modern English books unless they have one
4. **edition** - "First Edition", "Oxford World's Classics", etc.
5. **year** - Publication year of cited edition
6. **original_year** - Composition era if different (e.g., `-1200` for Enuma Elish, `800` for Beowulf). Use `null` if same as year
7. **publisher** - Full publisher name
8. **url** - If available online (archive.org, sacred-texts.com, etc.). Use `null` if not available. NEVER use banned URLs
9. **isbn** - ISBN string or `null` if pre-1967/ancient text
10. **source_type** - ancient | research | channeled | controversial
11. **citation_status** - provisional | locked
12. **astronomical_component** - unspecified | P | B (see below)
13. **component_reasoning** - One sentence explaining the component choice
14. **PLUS** either (`location_hint` + `summary`) OR (`page` + `quote`) based on citation_status
